## ✨ Features

- **Beautiful Generation**: Create stunning QR codes with custom colors, gradients, module shapes, and eye corner styles.
- **Brand Ready**: Embed your custom logos directly into the QR code with smart padding and layout calculations.
- **Advanced Inspection**: Decode and inspect any existing QR code. View error correction levels, version numbers, and the raw reconstructed module matrix.
- **Lightning Fast**: Built entirely with React, Vite, and Tailwind CSS.
- **Offline & History**: Your recent QR codes and settings are automatically saved in your browser for easy retrieval.
- **Export Anywhere**: Download your creations as high-resolution PNG, JPG, WebP, SVG vectors, or even PDF documents.

## 🚀 Quick Start

Ensure you have Node.js installed, then run the following commands:

```bash
# Clone the repository
git clone https://github.com/sam-eer31/qrcode_generator.git
cd qrcode_generator

# Install dependencies
npm install

# Start the development server
npm run dev
```

The app will be running at `http://localhost:5173`.

## 🛠 Tech Stack

- **Framework**: React 18 & TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS v4 & Framer Motion
- **Icons**: Lucide React
- **QR Engine**: `qrcode` (Generation) & `jsQR` (Decoding)
